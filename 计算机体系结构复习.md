3. Amdahl的传统定义中的计算机属性：
	+ 指令系统
	+ 数据表示
	+ 寻址规则
	+ 寄存器定义
	+ 中断系统
	+ 机器工作状态的定义和切换
	+ 存储系统
	+ 信息保护
	+ I/O结构
4. Flynn分类法是按照指令流和数据流的多重性进行分类
5. Flynn把计算机体系结构分为以下4类：
	+ 单指令流单数据流：（传统顺序处理计算机）
	+ 单指令流多数据流：（阵列计算机为代表）
	+ 多指令流单数据流
	+ 多指令流多数据流
6. SIMD（单指令多数据流）以阵列计算机为代表，在SIMD计算机中，在同一控制部件的控制下，多个处理部件同时执行同一条指令所规定的操作，分别对各自的数据进行处理。
